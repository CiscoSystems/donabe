--- layout: default title: About the Project ---

Donabe is a simple virtual application infrastructure deployment service for OpenStack. It was developed alongside the Curvature Advanced Dashboard project with the aim of changing the way that users interact with cloud-based virtual environments.

It is based around the concept of designing re-usable virtual application components from a set of OpenStack virtual networking and compute components. These components or "containers" can then be used to design increasingly complex application topologies by nesting them recursively within each other.

More details about the design and usage of Donabe can be found on the OpenStack Foundation website here.
